# 滑动窗口模板：最短子串覆盖所有字符种类

## 题型特征
- 给定一个字符串/数组
- 要找一个**最短连续子区间**
- 这个区间需要满足某种“覆盖条件”
  - 例如：包含原串中所有出现过的字符种类
  - 或包含某些目标字符集合

---

## 核心思路
使用双指针维护一个窗口 `[l, r]`：

- 右指针 `r` 不断右移，扩大窗口，让窗口逐渐满足条件
- 当窗口满足条件后，左指针 `l` 不断右移，缩小窗口
- 在窗口满足条件时更新答案

---

## 变量含义

```cpp
need
```
表示目标需要包含多少种字符。

```cpp
have
```
表示当前窗口中已经包含了多少种目标字符。

```cpp
cnt[ch]
```
表示当前窗口中字符 `ch` 出现了多少次。

```cpp
ans
```
记录满足条件的最短窗口长度。

---

## 通用流程

```text
1. 统计目标字符种类数 need

2. 初始化：
   l = 0
   have = 0
   ans = n

3. 枚举右端点 r：
   - 将 s[r] 加入窗口
   - 如果 s[r] 是窗口中新出现的字符，have++

4. 当窗口满足条件 have == need：
   - 更新答案 ans
   - 删除左端点 s[l]
   - 如果删除后某字符数量变为 0，have--
   - l++

5. 输出 ans
```

---

## C++ 模板

```cpp
int n;
cin >> n;

string s;
cin >> s;

vector<int> total(128, 0);
int need = 0;

for (char ch : s) {
    if (total[ch] == 0) {
        need++;
    }
    total[ch]++;
}

vector<int> cnt(128, 0);
int have = 0;
int l = 0;
int ans = n;

for (int r = 0; r < n; r++) {
    char ch = s[r];

    if (cnt[ch] == 0) {
        have++;
    }
    cnt[ch]++;

    while (have == need) {
        ans = min(ans, r - l + 1);

        char leftChar = s[l];
        cnt[leftChar]--;

        if (cnt[leftChar] == 0) {
            have--;
        }

        l++;
    }
}

cout << ans << "\n";
```

---

## 易错点
- [ ] 题目是否区分大小写？如果区分，`A` 和 `a` 是不同字符
- [ ] `have++` 只能在该字符第一次进入窗口时加
- [ ] 删除左端点后，如果计数变成 0，要 `have--`
- [ ] 更新答案要放在删除左端点之前
- [ ] 窗口长度是 `r - l + 1`

---

## 复杂度
每个指针最多移动 `n` 次：

```text
时间复杂度：O(n)
空间复杂度：O(字符集大小)
```
