## 🎯 双指针配对问题通用模板（最大化配对数量）

### ✅ **适用条件（必须同时满足）**
- 需要**最大化配对数量**
- 每个元素**只能使用一次**
- 配对条件**仅依赖元素大小关系**（如 `a_j ≥ k·a_i`）
- 数组**可排序**且排序后性质不变

### 🧠 **核心思想**
```
排序 → 分割两半 → 贪心匹配
```
- **排序**：统一大小顺序
- **分割**：前半部分作"小数"，后半部分作"大数"
- **贪心**：用最小的小数匹配最小的满足条件的大数

### 💻 **通用代码模板**
```cpp
#include<bits/stdc++.h>
#define endl "\n"
using ll = long long;
using namespace std;

int main()
{
    ll n;
    cin >> n;
    vector<ll> arr(n);
    for (int i = 0; i < n; i++)
    {
        cin >> arr[i];
    }

    // 1️⃣ 排序数组
    sort(arr.begin(), arr.end());
    
    // 2️⃣ 边界处理
    if (n < 2) 
    {
        cout << 0 << endl;
        return 0;
    }

    // 3️⃣ 关键：分割数组
    ll mid = n / 2;
    ll l = 0;        // 指向前半部分（小数候选）
    ll r = mid;      // 指向后半部分（大数候选）
    ll pairs = 0;    // 配对计数
    
    // 4️⃣ 双指针贪心匹配
    while (l < mid && r < n) 
    {
        // 🔄 根据题目修改此条件（示例：a_j ≥ 2·a_i）
        if (arr[r] >= 2 * arr[l]) 
        {
            pairs++;  // 配对成功
            l++;      // 小数已用
            r++;      // 大数已用
        } 
        else 
        {
            r++;  // 当前大数不满足，寻找更大的
        }
    }
    
    // 5️⃣ 输出结果
    cout << pairs << endl;
    return 0;
}
```

### ⚠️ **关键注意事项**
1. **分割点**：`mid = n/2` 是核心，不可省略
2. **指针范围**：`l < mid && r < n` 防止越界
3. **条件修改**：根据题目要求调整 `if` 判断条件
4. **移动逻辑**：配对成功时两个指针**同时移动**
5. **排序必要性**：不排序会导致贪心策略失效

### 🔁 **常见变体调整**
| 问题变化 | 代码调整点 |
|----------|------------|
| `a_j ≥ 3·a_i` | 修改 `if (arr[r] >= 3 * arr[l])` |
| `a_j ≤ k·a_i` | 反向遍历或调整指针方向 |
| 最小化配对 | 改用其他策略（非本模板） |
| 奇数长度 | `mid = (n+1)/2` 或保持 `n/2` |

### 📊 **性能指标**
- **时间复杂度**：O(n log n)（排序主导）
- **空间复杂度**：O(n)
- **Codeforces难度**：1300-1800
- **正确率**：正确实现时 >95%
