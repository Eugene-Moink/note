# 📝 凯撒密码算法笔记

## 🎯 **基本概念**
- **定义**：将字母表中的每个字母按照固定位数进行移位的加密方法
- **循环规则**：`z` 后面是 `a`，`Z` 后面是 `A`
- **输入限制**：通常只处理字母，其他字符保持不变

---

## 🔑 **核心公式**

| 操作 | 公式 | 说明 |
|------|------|------|
| **加密** | `ch = 'a' + ((s[i]-'a') + n) % 26` | 小写字母向后移n位 |
| **解密** | `ch = 'a' + ((s[i]-'a') - n + 26) % 26` | 小写字母向前移n位 |
| **大写加密** | `ch = 'A' + ((s[i]-'A') + n) % 26` | 大写字母向后移n位 |

---

## 🧮 **算法步骤**

### **小写字母处理**
```cpp
// 1. 转换为索引(0~25)
int idx = s[i] - 'a';

// 2. 向后移动n位并循环
int new_idx = (idx + n) % 26;

// 3. 转回字符
char result = 'a' + new_idx;
```

### **完整字符判断**
```cpp
if(s[i] >= 'a' && s[i] <= 'z') {  // 小写
    s[i] = 'a' + ((s[i] - 'a') + n) % 26;
}
else if(s[i] >= 'A' && s[i] <= 'Z') {  // 大写
    s[i] = 'A' + ((s[i] - 'A') + n) % 26;
}
else {  // 其他字符不变
    // 不处理
}
```

## 🛠️ **代码模板**

### **基础模板**
```cpp
#include<bits/stdc++.h>
#define endl "\n"
using ll = long long;
using namespace std;

int main()
{
    ll n;
    string s;
    cin >> n >> s;
    
    for(int i = 0; i < s.size(); i++)
    {
        if(s[i] >= 'a' && s[i] <= 'z')  // 小写
            s[i] = 'a' + ((s[i] - 'a') + n) % 26;
        else if(s[i] >= 'A' && s[i] <= 'Z')  // 大写
            s[i] = 'A' + ((s[i] - 'A') + n) % 26;
    }
    
    cout << s << endl;
    return 0;
}
```

### **函数封装版**
```cpp
char caesar_shift(char c, int shift, bool forward = true) 
{
    if(c >= 'a' && c <= 'z') {
        int dir = forward ? shift : -shift;
        return 'a' + ((c - 'a') + dir + 26) % 26;
    }
    else if(c >= 'A' && c <= 'Z') {
        int dir = forward ? shift : -shift;
        return 'A' + ((c - 'A') + dir + 26) % 26;
    }
    return c;  // 非字母不变
}
```

## ⚠️ **注意事项**

| 项目 | 说明 |
|------|------|
| **模运算** | 必须用 `% 26` 处理循环 |
| **负数修正** | `(x % 26 + 26) % 26` 防止负数 |
| **字符范围** | 用 `islower()` `isupper()` 判断 |
| **大数移位** | `n % 26` 自动处理 `n > 26` |
| **边界测试** | `'z' + 1 = 'a'`, `'a' - 1 = 'z'` |

## 🧪 **测试用例**

| 输入 | 输出 | 说明 |
|------|------|------|
| `n=1, s="abc"` | `"bcd"` | 正常移位 |
| `n=1, s="xyz"` | `"yza"` | 循环处理 |
| `n=25, s="abc"` | `"zab"` | 大移位量 |
| `n=1, s="ABc!"` | `"BCd!"` | 大小写混合 |

## 📊 **复杂度分析**

| 类型 | 复杂度 | 说明 |
|------|--------|------|
| **时间** | O(n) | n为字符串长度 |
| **空间** | O(1) | 原地修改 |
| **适用范围** | 字符串长度≤10⁶ | 适合大部分场景 |
