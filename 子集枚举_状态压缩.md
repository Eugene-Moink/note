
## 题型名称

1) **子集枚举 / 状态压缩（Bitmask / 状压）**
2) 也常被归到 **暴力搜索 / 回溯（Brute force / DFS）**（当 n 很小）

本题 PERKET 的典型标签就是：**n 很小 → 枚举所有非空子集 → 计算目标函数取最小**。

---

## 以后遇到这类题怎么做（通用套路）

### Step 1：先看 n 小不小

- 如果 \(n \le 20\)（甚至更小如 10）：优先想 **枚举所有子集**子集数 \(2^n\)：
  - n=10 → 1024（秒过）
  - n=20 → ~1,048,576（也常能过，注意常数）

### Step 2：用 mask 表示“选/不选”

- `mask` 的第 i 位 = 1：选第 i 个
- 第 i 位 = 0：不选第 i 个
- 枚举范围：
  - 全部子集：`mask = 0 .. (1<<n)-1`
  - 非空子集：`mask = 1 .. (1<<n)-1`

判断是否选择第 i 个：

```cpp
if (mask & (1<<i)) { ... }
```

### Step 3：对每个 mask 计算你要的量

常见两类：

- 求和型：`sum += val[i]`
- 乘积/最大值/最小值等：按题意更新
  PERKET 就是：
- `prod *= s[i]`
- `sum += b[i]`
- `ans = min(ans, abs(prod - sum))`

### Step 4：注意“空集是否合法”

题目常写：

- “至少选一个” → 跳过 mask=0（从 1 开始）
- “可以不选” → mask=0 也要算

### Step 5：常见坑

- 下标混乱（0/1 下标）
- `1<<n` 的类型（习惯写 `1LL<<n` 更安全）
- 乘积初值要用 1（不是 0）
- `abs` 用 long long 时写 `llabs` 或确保转成 `long long`

---

## 一句话记忆法

> **n 小 + 每个元素“选/不选” → 用 bitmask 枚举所有子集 → 每个子集算答案取最优。**
