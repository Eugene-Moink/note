# 区间加法 + 最值查询（差分数组）模板笔记

## 题型特征
- 有初始数组 `a[1..n]`
- 有 `p` 次操作：对区间 `[l, r]` 的每个元素都加上 `val`
- 最后需要输出某种统计量：最小值/最大值/最终数组等
- 朴素写法每次操作循环更新 `[l..r]` 会 TLE（最坏 O(n*p)）

---

## 核心思想：用差分把“区间更新”变成“两个点更新”
维护“增量数组”的差分 `d`：

对一次操作 `[l, r] += val`：
- `d[l] += val`
- `d[r+1] -= val`

然后一次扫描用前缀和还原每个位置的总增量：
- `add += d[i]`  （此时 `add` 就是位置 i 应加的总量）
- `final[i] = a[i] + add`

整体复杂度：`O(n + p)`

---

## 为什么是 r+1？
因为差分前缀和表示“从某个点开始一直生效”：
- 在 `l` 开始生效：`+val`
- 在 `r` 结束，所以从 `r+1` 开始取消：`-val`

---

## 模板（1-based，下标最常用）
```cpp
int n, p;
cin >> n >> p;
vector<long long> a(n + 1);
vector<long long> d(n + 2, 0);   // 必须 n+2，避免访问 r+1 越界

for (int i = 1; i <= n; i++) cin >> a[i];

while (p--) {
    int l, r;
    long long val;
    cin >> l >> r >> val;
    d[l] += val;
    d[r + 1] -= val;
}

long long add = 0;
long long mn = LLONG_MAX;   // 若求最小值
long long mx = LLONG_MIN;   // 若求最大值

for (int i = 1; i <= n; i++) {
    add += d[i];
    long long cur = a[i] + add;  // 还原后的最终值

    mn = min(mn, cur);
    mx = max(mx, cur);
}

// 按题目需要输出 mn 或 mx 或整段 final
cout << mn << "\n";
