## 基础概念

矩阵旋转是指将二维矩阵按照一定角度进行旋转的操作。在算法竞赛中，最常见的是90度倍数的旋转。

对于一个 $n \times m$ 的矩阵：
- 原矩阵大小：$n$ 行 $m$ 列
- 旋转90度后：$m$ 行 $n$ 列（行列数量互换）

## 旋转公式

### 顺时针旋转90度
```
原矩阵(i,j) → 新矩阵(j, n-1-i)
```

### 逆时针旋转90度
```
原矩阵(i,j) → 新矩阵(m-1-j, i)
```

### 旋转180度
```
原矩阵(i,j) → 新矩阵(n-1-i, m-1-j)
```

### 旋转270度（顺时针）/ 90度（逆时针）
```
原矩阵(i,j) → 新矩阵(m-1-j, i)
```

## 代码模板

### 通用旋转类
```cpp
class MatrixRotation {
public:
    // 顺时针旋转90度
    static vector<vector<int>> rotate90Clockwise(const vector<vector<int>>& matrix) {
        int n = matrix.size();
        int m = matrix[0].size();
        vector<vector<int>> result(m, vector<int>(n));
        
        for(int i = 0; i < n; i++) {
            for(int j = 0; j < m; j++) {
                result[j][n-1-i] = matrix[i][j];  // 核心公式
            }
        }
        return result;
    }
    
    // 逆时针旋转90度
    static vector<vector<int>> rotate90CounterClockwise(const vector<vector<int>>& matrix) {
        int n = matrix.size();
        int m = matrix[0].size();
        vector<vector<int>> result(m, vector<int>(n));
        
        for(int i = 0; i < n; i++) {
            for(int j = 0; j < m; j++) {
                result[m-1-j][i] = matrix[i][j];  // 核心公式
            }
        }
        return result;
    }
    
    // 旋转k度（k为90的倍数）
    static vector<vector<int>> rotateK(const vector<vector<int>>& matrix, int k) {
        k = ((k % 360) + 360) % 360;
        k /= 90;
        
        vector<vector<int>> result = matrix;
        for(int i = 0; i < k; i++) {
            result = rotate90Clockwise(result);
        }
        return result;
    }
};
```

### 完整实现
```cpp
#include<bits/stdc++.h>
#define endl "\n"
using ll = long long;
using namespace std;

int main() {
    int n, m;
    cin >> n >> m;
    
    vector<vector<int>> matrix(n, vector<int>(m));
    for(int i = 0; i < n; i++) {
        for(int j = 0; j < m; j++) {
            cin >> matrix[i][j];
        }
    }
    
    // 顺时针旋转90度
    vector<vector<int>> rotated(90, vector<int>(n));
    for(int i = 0; i < n; i++) {
        for(int j = 0; j < m; j++) {
            rotated[j][n-1-i] = matrix[i][j];
        }
    }
    
    // 输出结果
    for(int i = 0; i < m; i++) {
        for(int j = 0; j < n; j++) {
            cout << rotated[i][j];
            if(j != n-1) cout << " ";
        }
        cout << endl;
    }
    
    return 0;
}
```

## 常见错误

### ❌ 错误1：坐标变换公式记错
```cpp
// 错误写法
new_matrix[k][n-1-l] = old_matrix[k][l];  // 错误：k和l的对应关系不对
```

### ✅ 正确写法
```cpp
// 正确写法
new_matrix[l][n-1-k] = old_matrix[k][l];  // (k,l) → (l, n-1-k)
```

## 时间复杂度
- **时间复杂度**: $O(n \times m)$
- **空间复杂度**: $O(n \times m)$

其中 $n$ 和 $m$ 分别为原矩阵的行数和列数。
