
这题属于很典型的 **“线性不定方程计数 / 枚举一种变量”**（也可叫“鸡兔同笼类”）问题：给定总量，用若干种固定贡献（2、4）去凑，统计非负整数解个数。

下面给你一个可复用的“模板思路”，以后碰到类似题基本直接套。

---

## 题型名称

- **非负整数解计数**（linear diophantine counting）
- 竞赛常见别名：**鸡兔同笼 / 枚举法计数**

---

## 通用模板（适用于：\(a\cdot x + b\cdot y = n\)，求 \(x,y\ge0\) 解的个数）

### Step 1：选一个变量枚举（优先枚举系数更大的那个）

例如本题：\(2c + 4w = n\)枚举牛 \(w\)（因为 4 大，枚举次数少）：

- \(w = 0.. \lfloor n/4 \rfloor\)
- 剩余：`rem = n - 4*w`
- 要求 `rem >= 0` 且 `rem % 2 == 0`
  - 满足则鸡数 `c = rem/2` 是非负整数 → 计数 +1

### Step 2：输出计数即可

复杂度：\(O(n/\max(a,b))\)，n 很小或中等都稳。

---

## 本题专用的精简模板

```cpp
cnt = 0
for w in [0..n/4]:
    rem = n - 4*w
    if rem % 2 == 0: cnt++
print cnt
```

（因为 rem 自动非负）

---

## 你以后套模板时要检查的 4 个点（防错清单）

1) **枚举范围**：`0.. n/coef`（一定从 0 开始，因为允许为 0）
2) **剩余量非负**：通常 `rem = n - coef*var`，枚举上界保证 rem≥0
3) **整除条件**：`rem % other_coef == 0`
4) **非空/其它限制**：如果题目要求至少选一种、或者上限限制，要额外判断

---

## 什么时候不能只枚举？

如果有 3 种以上物品（3 个变量），就可能需要：

- 双重枚举 + 判断
- 或 DP（背包）
- 或数论推导

但“两种物品 + 一个总和/总量”的，几乎都能用上面一层枚举搞定。
