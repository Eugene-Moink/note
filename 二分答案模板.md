# 二分答案模板笔记

## 适用场景
- **最值优化问题**：求最大值中的最小值、最小值中的最大值
- **存在单调性**：若 x 满足条件，则 x±1 也存在单调关系
- **可验证性**：给定值能在合理时间内验证是否符合条件

## 核心模板

```cpp
// 求满足条件的最大值
ll left = 最小可能值, right = 最大可能值;
ll ans = 0;

while (left <= right) {
    ll mid = left + (right - left) / 2;
    
    if (check(mid)) {           // 检查mid是否满足条件
        ans = mid;              // 记录可行解
        left = mid + 1;         // 继续寻找更大值
    } else {
        right = mid - 1;        // 当前值不满足，向小值搜索
    }
}

// 求满足条件的最小值
ll left = 最小可能值, right = 最大可能值;
ll ans = 最大可能值;

while (left <= right) {
    ll mid = left + (right - left) / 2;
    
    if (check(mid)) {
        ans = mid;              // 记录可行解
        right = mid - 1;        // 继续寻找更小值
    } else {
        left = mid + 1;         // 当前值不满足，向大值搜索
    }
}
```

## 关键注意事项

### 1. 防溢出技巧
```cpp
// 错误：可能溢出
if (mid * mid * mid <= n)

// 正确：防止溢出
if (mid <= n / mid / mid)
```

### 2. 搜索边界
- `left = mid + 1` vs `left = mid`
- `right = mid - 1` vs `right = mid`
- 关键在于 mid 是否可能是答案

### 3. 初始化策略
- `ans` 初始值应为"无效值"
- 搜索范围要覆盖所有可能性

## 快速判断方法
遇到问题时问自己：
1. **是否求最值？** → 最大值/最小值
2. **是否存在单调性？** → x满足→x±1的规律性
3. **能否快速验证？** → check函数效率

全部是 → 使用二分答案！

## 典型变体
- **最大化最小值**：最小值尽可能大
- **最小化最大值**：最大值尽可能小
- **第k大/小**：在有序序列中定位

记住：**单调性 + 验证函数 + 二分搜索 = 二分答案**
