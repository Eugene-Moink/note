# 最长连续段（Longest Consecutive Segment）模板

## 适用题型

- 求数组/字符串中满足某条件的**最长连续**区间长度
  例：最长连续 0、最长连续某字符、最长连续偶数等

---

## 核心变量

- `cur`：当前连续段长度
- `best`：历史最大连续段长度

---

## 扫描模板（一次遍历 O(n)）

```text
cur = 0
best = 0
for each element x:
    if ok(x):        // 当前元素满足条件
        cur++
    else:            // 条件断开
        best = max(best, cur)
        cur = 0
best = max(best, cur) // 处理结尾仍在连续的情况
输出 best
```
