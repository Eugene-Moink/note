# 一维扫描 + 贪心跳跃（最少操作次数）模板

## 适用特征
- 对一维序列/字符串进行操作  
- 每次操作影响一段连续区间（常见固定长度 `k`）  
- 目标是消除/覆盖所有“坏点”（如字符 `B`）  
- 求最少操作次数，且操作之间不需要复杂联动

---

## 核心思路（不变式）
从左到右扫描，始终保证：`[0, i)` 区间已经被“处理完”，不再需要回头。

- 若当前位置是“好点”：`i += 1`
- 若当前位置是“坏点”：必须做一次操作覆盖它  
  - `ans += 1`
  - `i += jump`（通常 `jump = k`，直接跳过本次操作覆盖的区间）

---

## 伪代码模板
```text
ans = 0
i = 0
while i < n:
    if is_bad(i):
        ans++
        i += jump   // jump 通常为 k
    else:
        i++
输出 ans
```

---

## C++ 代码骨架

```cpp
int ans = 0;
for (int i = 0; i < n; ) {
    if (/* is_bad(s[i]) */) {
        ans++;
        i += k;      // 或者按题意的 jump
    } else {
        i++;
    }
}
cout << ans << "\n";
```

---

## 常见坑点检查清单

- [ ] 循环写成 `for (i=0; i<n; )` / `while`，不要固定 `i++`
- [ ] 每个测试用例 `ans` 要清零
- [ ] 别忘了输出 `ans`
- [ ] 注意 `n` 总和大时加快 IO：
  `ios::sync_with_stdio(false); cin.tie(nullptr);`

---

## 自测边界

- 全是好点：答案应为 `0`
- `k = 1`：答案通常等于坏点数量
- 坏点在末尾：仍应能被一次操作覆盖（实现上直接 `i += k` 一般没问题）
