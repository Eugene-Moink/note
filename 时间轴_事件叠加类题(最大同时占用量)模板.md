# 时间轴/事件叠加类题（最大同时占用量）模板：差分 + 前缀和

## 题型特征
- 有很多个任务/区间，每个区间 `[s, t)` 或 `[s, t]` 内会“占用”一定资源 `w`
- 问：最少需要多少资源才能保证不冲突  
  等价于：**任意时刻同时占用的资源总和的最大值**

典型例子：桶/教室/机器/带宽/人流同时在线等

---

## 核心思路
1. 把每个区间看成对时间轴的“区间加法”
2. 用差分数组 `d` 做区间加：每条记录只更新两个点
3. 一次前缀和扫时间，得到每个时刻的总占用量 `cur`
4. 取最大值 `ans = max(cur)`

---

## 区间端点怎么选？（最容易错）
- 若任务在 **[s, t)** 生效（从 s 开始，t 时刻已结束不占用）：
  - `d[s] += w`
  - `d[t] -= w`

- 若任务在 **[s, t]** 生效（包含 t）：
  - `d[s] += w`
  - `d[t+1] -= w`

> USACO/很多题中常用 `[s, t)`（结束时刻不占用）。

---

## C++ 通用模板（[s, t) 版本）
```cpp
int N;
cin >> N;

const int T = 1000;              // 时间上限（按题目给的最大时间）
vector<long long> d(T + 2, 0);   // 至少开到 T+1，方便访问 d[t]

for (int i = 0; i < N; i++) {
    int s, t;
    long long w;
    cin >> s >> t >> w;
    d[s] += w;
    d[t] -= w;                   // [s, t) 模式
}

long long cur = 0, ans = 0;
for (int time = 1; time <= T; time++) {
    cur += d[time];
    ans = max(ans, cur);
}

cout << ans << "\n";
