# 通用兑换问题求解模板

## 问题描述
给定一个兑换规则：用 `exchange_need` 个收集品可以兑换 `exchange_get` 个新物品，每个物品消费后产生1个收集品。求要获得 `target` 个物品，最少需要初始购买多少个物品。

## 模板代码

```cpp
#include<bits/stdc++.h>
using ll = long long;
using namespace std;

/**
 * 通用兑换函数
 * @param initial_count: 初始购买/获得的数量
 * @param exchange_need: 需要多少个收集品才能兑换1个新物品
 * @param exchange_get: 1次兑换能获得多少个新物品
 * @return: 总共能获得的最大物品数
 */
ll maxExchange(ll initial_count, ll exchange_need, ll exchange_get = 1) {
    ll total_obtained = initial_count;    // 总共获得的物品数
    ll collectibles = initial_count;      // 当前拥有的收集品数量
    
    while (collectibles >= exchange_need) {  // 收集品足够兑换就继续
        ll exchanges = collectibles / exchange_need;  // 这次能兑换多少次
        ll new_items = exchanges * exchange_get;      // 获得的新物品数
        total_obtained += new_items;                  // 加入总获得数
        
        collectibles = collectibles % exchange_need + new_items;  // 更新收集品数
        // collectibles % exchange_need : 兑换后剩余的收集品
        // + new_items : 新获得物品消费后产生的新收集品
    }
    
    return total_obtained;
}

/**
 * 二分查找最小满足条件的初始值
 * @param target: 目标总数
 * @param exchange_need: 兑换所需收集品数量
 * @param exchange_get: 单次兑换获得物品数量
 * @return: 最小的初始购买数量
 */
ll minInitialForTarget(ll target, ll exchange_need, ll exchange_get = 1) {
    ll left = 1, right = target;
    
    while (left < right) {
        ll mid = left + (right - left) / 2;
        if (maxExchange(mid, exchange_need, exchange_get) >= target) {
            right = mid;      // 满足条件，尝试更小值
        } else {
            left = mid + 1;   // 不满足条件，必须增大
        }
    }
    
    return left;
}

int main() {
    ll target;          // 目标获得总数
    ll exchange_need;   // 兑换所需收集品数量
    ll exchange_get;    // 单次兑换获得物品数量（默认为1）
    
    cin >> target >> exchange_need;
    cin >> exchange_get;  // 如果不输入，默认为1
    
    ll result = minInitialForTarget(target, exchange_need, exchange_get);
    cout << result << endl;
    
    return 0;
}
```

## 使用说明

### 1. 基本参数
- `target`: 你需要达到的目标总数
- `exchange_need`: 需要用多少个收集品才能兑换
- `exchange_get`: 一次兑换能获得多少个新物品（通常为1）

### 2. 使用步骤

#### 步骤1：确定参数
根据题目要求确定三个关键参数：
- `target`: 最终想要获得的物品总数
- `exchange_need`: 兑换门槛（如：3个空瓶换1瓶酒）
- `exchange_get`: 兑换收益（如：1个空瓶换1瓶酒）

#### 步骤2：调用函数
- 如果想知道买 `x` 个能获得多少个，调用 `maxExchange(x, exchange_need, exchange_get)`
- 如果想知道要获得 `target` 个最少买多少个，调用 `minInitialForTarget(target, exchange_need, exchange_get)`

### 3. 典型应用场景

#### 场景1：冰棍问题
- `target = 10`, `exchange_need = 3`, `exchange_get = 1`
- 意思：想吃10根冰棍，3个木棒换1根，问最少买几根

#### 场景2：饮料问题  
- `target = 15`, `exchange_need = 4`, `exchange_get = 1`
- 意思：想喝15瓶饮料，4个空瓶换1瓶，问最少买几瓶

#### 场景3：积分兑换
- `target = 100`, `exchange_need = 8`, `exchange_get = 3` 
- 意思：要获得100个礼品，8积分换3个礼品，问最少需要多少积分

### 4. 注意事项
- 所有参数都应该是正整数
- `exchange_get` 应该小于 `exchange_need`，否则会出现无限兑换的情况
- 时间复杂度为 O(log² n)，适合处理大数据

### 5. 自定义修改
如果题目有特殊规则（如部分兑换、不同阶段不同兑换率等），可以在 `maxExchange` 函数内部添加相应的判断逻辑。
